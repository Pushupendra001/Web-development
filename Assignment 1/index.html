Moto Racing Game

Coins: 0 Lifelines: 3

Game Overview

This is a moto racing game where you can control the speed and lane of your bike. Press the spacebar to increase the speed, and use the arrow keys to switch lanes.

Game Highlights

Experience the thrill of racing through a dynamically changing track with obstacles.

You have three lifeline, if you collide with another vehicle you will lose a lifeline.

While playing game you will find find some coin as reward.

Use F keyword for Full screen and esc for escape.

How to Get Started

Just open the game and start playing! Use the spacebar to accelerate and the arrow keys to switch lanes.
